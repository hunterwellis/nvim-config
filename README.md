# neovim config

Configuration files for my personal neovim configuration (based on NvChad).

![neovim screenshot](./images/screenshot.png)

## dependencies
- nvim
- npm
- yarn

## install
1. ```git clone https://github.com/hunterwellis/nvim-config ~/.config/nvim && nvim```
2. ```:Lazy sync```
3. ```:MasonInstallAll```
2. ```:MasonInstallAll```
3. ```:Lazy sync```
